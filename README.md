# 保质期管理网页应用

这是一个纯前端的保质期管理工具，使用浏览器本地存储（localStorage）保存数据，无需后端。

## 功能简介

- 记录物品：名称、类别、生产日期、到期日期、提前提醒天数
- 自动计算：剩余天数、状态（正常 / 即将过期 / 已过期）
- 支持搜索、状态筛选、排序
- 支持导出 / 导入 JSON 备份数据

## 使用方式

### 直接打开

1. 双击打开 `index.html`（推荐使用 Chrome / Edge）
2. 开始新增物品并管理保质期

### 本地启动静态服务器（可选）

如果你安装了 Node.js，可以在项目根目录下运行：

```bash
npm install
npm run start
```

然后在浏览器访问终端里提示的本地地址。
